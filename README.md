## ✨ Fitur Utama

### 1. **Manajemen Menu**
   - Tambah, edit, dan hapus item menu kafe
   - Kategorisasi produk (Kopi, Minuman, Makanan, Dessert)
   - Pengelolaan harga dan stok

### 2. **Sistem Pesanan**
   - Pencatatan pesanan real-time
   - Tracking status pesanan (Pending, Proses, Siap, Selesai)
   - Integrasi dengan sistem pembayaran

### 3. **Manajemen Inventory**
   - Pantau stok bahan baku dan produk jadi
   - Notifikasi stok minimum otomatis
   - Laporan perputaran inventory

### 4. **Laporan & Analitik**
   - Dashboard penjualan harian/bulanan
   - Analisis produk terlaris
   - Laporan keuangan dan keuntungan

### 5. **Manajemen Karyawan**
   - Data karyawan dan jadwal kerja
   - Pencatatan absensi
   - Manajemen gaji dan bonus

## 📦 Instalasi

### Prasyarat
- [Teknologi yang diperlukan - contoh: Node.js v14+, Python 3.8+, dll]
- Git

### Langkah Instalasi

1. **Clone Repository**
   ```bash
   git clone https://github.com/fahmsi/PROJEK00-UTS.git
   cd PROJEK00-UTS
   ```

2. **Instalasi Dependensi**
   ```bash
   # Jika menggunakan npm
   npm install
   
   # Atau jika menggunakan pip
   pip install -r requirements.txt
   ```

3. **Konfigurasi Database**
   - Update file konfigurasi dengan kredensial database Anda
   - Jalankan migrasi database

4. **Jalankan Aplikasi**
   ```bash
   npm start
   # atau
   python app.py
   ```

5. **Akses Aplikasi**
   ```
   http://localhost:[PORT]
   ```
